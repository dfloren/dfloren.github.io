Boggle Solver
From Wikipedia: "Boggle is a word game in which players try to find as many words as they can from a grid of lettered dice, within a set time limit. It was invented by Allan Turoff and originally distributed by Parker Brothers. The game is played using a plastic grid of lettered dice, in which players look for words in sequences of adjacent letters."
